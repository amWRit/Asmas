This feature is available under Tools>View>View My Classes

A class teacher will be able to use this feature.

Right click on the list for more OPTIONS.

  1. View

    View details of the class

  2. Delete

    Delete the class

  3. View Students

    View students of that class Right click for more options on the list : View, Edit, Delete

  4. Add student

    Add a student form students database to your class

  5. View results

    You can filter the results by terminal, update the calculations of the results, update rank, print results. Right click for more options : EDIT

  6. Add result

    Student Wise -Add result of all subjects of a student Subject Wise -Add result of a subject of all the students at once

  7. Subject Teacher

    You can view subject teachers of your class. You can Assign a subject teacher. You can analyse the result subject wise; get average, high, low and LOCOS. - Subject Result Analysis -- Right click for more options: View sutdents under LOCOS.
