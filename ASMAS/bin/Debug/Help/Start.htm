You are the ADMIN.

These are the basic things you should do before using other functions of the program.

Go to Admin Tools menu and do the following..

  1. First add a Year, possibly the current year.
  2. Then add a School for that year.
  3. Then add USERS.

    Users are basically the teachers in your school. Users can be an admin, class teacher or a viewer. If you want to create a user that can only view the database, create a user with role 'viewer'.

  4. Now create Classes.

    You can choose the class teacher of the class based on users you just created.

  5. Now CREATE students.
  6. You have now created the basic database.
  • Database Import/Export

    You can export the current database to an access file.

    You can import the database from an another access file. This will completely renew your database. So, use this tool carefully. A backup of the current database will be created.
